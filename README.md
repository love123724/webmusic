#前端技术栈

## 1.Vue全家桶+vue-cli3
  i.code.js
  ii.vue-router前端路由
  iii.vuex 状态管理
	
	
	
## 2.插件
i.better-scroll

ii.pinyin 将中文转拼音的api

iii.good-storage 本地存储api

iv.vue-lazyload 图片懒加载

v.fastclick

vi.postcss-px-to-viewport 适配
